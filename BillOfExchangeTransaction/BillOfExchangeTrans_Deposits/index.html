Bill Of Exchange Trans Deposits Object

Description

BillOfExchangeTrans_Deposits is a child object of the BillOfExchangeTransaction object and represents the deposits information for incoming payments.

Source table: ODPS.

Object Model

Fields
Field
ValidValuesValidValue

Remarks

Mandatory fields is SAP Business One: BankAccount (bank code), BankCountry, and BankDepositAccount.

To display the form in the application:

  1. Select Banking --> Bill of Exchange.
  2. Select one of the Bill of Exchange sub-menus Management or Fund. Set your criteria and then click OK.
    The ODPS table also relates to Banking --> Deposits --> Deposit --> Bill of Exchange tab.
