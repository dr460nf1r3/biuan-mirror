Get Blanket Agreement Method

Description

Retrieves a blanket agreement.

The blanket agreement is specified by its key, which is contained in the BlanketAgreementParams object passed to the method.

Syntax

Visual Basic
Public Function GetBlanketAgreement( _
   ByVal pIBlanketAgreementParams As BlanketAgreementParams _
) As BlanketAgreement

Parameters

pIBlanketAgreementParams

The key of the blanket agreement to retrieve.
