Branches Service

Description

The BranchesService service enables you to add, look up and remove branches in the branches master data table. Branches can be assigned to users and employees.

To see the list of branches and create a new one, select Administration --> Setup --> General --> Users, and then select the Branch field.

Source table: OUBR
