Specifies the business object that you want to create.

Return Type

The GetBusinessObject method returns the object type specified in the method's parameter. The returned object is empty and contains only the default values specified in the company database.

Remarks

You can use this method to create business objects such as, BusinessPartners, Items, and so on. Then, to use the created object (to call methods such as, Add, Update, GetByKey, and so on) you must set the appropriate values to the object's properties (including the mandatory properties).

To load an existing object from an XML file, call the GetBusinessObjectFromXML method.

To release an object after using it, use the code line:

Set object = Nothing
