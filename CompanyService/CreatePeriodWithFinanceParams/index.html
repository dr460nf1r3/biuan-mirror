Create Period With Finance Params Method

Description

Returns a PeriodCategoryParams Identification Key, extended with finance parameters derived by the FinancePeriodParams identification key (system number, period indicator).

Syntax

Visual Basic
Public Function CreatePeriodWithFinanceParams( _
   ByVal pIPeriodCategory As PeriodCategory, _
   ByVal pIFinancePeriodParams As FinancePeriodParams _
) As PeriodCategoryParams

Parameters

pIPeriodCategory
PeriodCategory data structure input.
pIFinancePeriodParams
FinancePeriodParams identification key (system number, period indicator) input.
