Currency Restrictions Object

Description

The CurrencyRestrictions is a child object of the WizardPaymentMethods object.

This object enables to allow or restrict currencies in the payment method.

Source table: PYM1.

Object Model

CurrencyRestrictions
UserFields
Fields
Field
ValidValuesValidValue

Remarks

To display the form in the application:

  1. Select Administration -->Setup -->Banking -->Payment Methods.
  2. Click the [...] button near the Currency Restriction field.

The CurrencyRestrictions object is applicable when the CurrencyRestriction property of the WizardPaymentMethods object is set to tYES.
