API Default Report Params
Default Report Params Object

Description

Specifies the default report layout for a document type. The default report layout can be designated for a specific user and business partner, or for all users and business partners.

Use this object with the GetDefaultReport/SetDefaultReport methods of the ReportLayoutsService.

Source table: RDFL

Remarks

To specify a default report layout in SAP Business One:

  1. Open a marketing document form (for example, Sales A/R -- Sales Quotation).
  2. Select Tools --> Print Layout Designer (or select the pencil icon on the toolbar).
  3. Select a report layout.
  4. Click Set as Default.
