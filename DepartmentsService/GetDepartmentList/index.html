Get Department List Method

Description

Retrieves the keys and names of all the departments.

Example

Getting a list of departments (C#)
DepartmentsParams getlistParams; getlistParams = oDeptSrv.GetDepartmentList(); String resultSet = ""; foreach (DepartmentParams record in getlistParams) { resultSet = resultSet + record.Code + "\t" + record.Name + "\n"; }
