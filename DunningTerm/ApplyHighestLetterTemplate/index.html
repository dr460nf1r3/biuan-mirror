Apply Highest Letter Template Property

Description

Indicates whether to use the template for the highest dunning level of all the documents included in the letter.

The default is no.

Field name: HiLtrFrmt

Property type

Read-write property

Syntax

Visual Basic
Public Property ApplyHighestLetterTemplate() As BoYesNoEnum

Remarks

Relevant only if the GroupingMethod property is set to per business partner (gmPerBP) or set per dunning level (gmPerDunningLevel).
