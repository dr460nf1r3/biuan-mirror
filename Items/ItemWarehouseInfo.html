Item Warehouse Info Object

Description

ItemWarehouseInfo is a child object of the Items object that represents the items in the warehouse. This object is part of the Inventory and Production module. This object enables you to add an item to the warehouse.

Source table: OITW.

Remarks

Mandatory fields in SAP Business One: WarehouseCode and ItemCode (Items object).

To display the form in the application:

  1. Select Inventory --> Item Master Data.
  2. Select Inventory Data tab.
