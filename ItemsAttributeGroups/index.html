Items Attribute Groups Object

Description

ItemsAttributeGroups is a child object of the Items object.

You can manage the attributes of an asset.

Source table: ITM13.

Remarks

To access the subtab, from the SAP Business One Main Menu, choose Financials --> Fixed Assets --> Asset Master Data. Select the Fixed Assets tab and then the Additional Information subtab.
