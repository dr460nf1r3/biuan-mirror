Journal Entries Lines Object

Description

Journal_Entries_Lines is a child object of the JournalEntries object, and represents the line entries of each transaction. This object enables you to add a journal transaction line.

Source table: JDT1.

Object Model

Remarks

Mandatory fields in SAP Business One: ShortName, and Credit or Debit, .

To display the form in the application:

  • Select Financials --> Journal Entry (view in Expand Editing Mode).
