Update Project Method

Description

Replaces project code and name with the specified Project data structure.

Syntax

Visual Basic
Public Sub UpdateProject( _
   ByVal pIProject As Project _
)

Parameters

pIProject
Specifies the code and name that will replace the current project.

Example

Update a Project (Visual Basic)
Dim oCmpSrv As SAPbobsCOM.CompanyService Dim projectService As SAPbobsCOM.IProjectsService Dim project As SAPbobsCOM.IProject Dim projectParams As SAPbobsCOM.IProjectParams oCmpSrv = oCompany.GetCompanyService projectService = oCmpSrv.GetBusinessService(SAPbobsCOM.ServiceTypes.ProjectsService) 'Get a project projectParams = projectService.GetDataInterface(SAPbobsCOM.ProjectsServiceDataInterfaces.psProjectParams) projectParams.Code = "PRJ11" project = projectService.GetProject(projectParams) 'Update the project project.Name = "NewProjectName" projectService.UpdateProject(project)
