Sales Forecast Object

Description

SalesForecast is a business object that represents the sales forecast for a specified period. The sales forecast is required for planning the purchase and production of items in the MRP module.

This object enables you to:

  • Add a sales forecast.
  • Retrieve a sales forecast by its key.
  • Update a sales forecast.
  • Remove a sales forecast.
  • Save the object in XML format.

Source table: OFCT.
