API Sales Stages
Sales Stages Object

Description

The SalesStages object enables defining sales stages and their probability percentage. For example: Lead, Meeting, Quotation, Negotiation, and Order. These definitions are used as default values for the SalesOpportunities object.

Source table: OOST.

Remarks

Mandatory fields in SAP Business One: Name and Stageno.

  • Select Administration -->Setup -->Sales Opportunities -->Sales Stages.
