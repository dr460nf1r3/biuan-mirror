Service Call Activities Object

Description

ServiceCallActivities is a child object of the ServiceCalls object in the Service module. The service call activities are associated with the Activities with Business Partners table.

Source table: SCL5.

Object Model

ServiceCallActivities

ValidValuesValidValue

Remarks

Mandatory field in SAP Business One: ActivityCode.

To display the form in the application:

  1. Select Service --> Service Call.
  2. Select the Activities tab.
