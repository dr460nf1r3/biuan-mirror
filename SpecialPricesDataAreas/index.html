Special Prices Data Areas Object

Description

SpecialPricesDataAreas is a child object of SpecialPrices object and represents special prices that are valid only for specified periods such as, holidays and season sales.

Source table: SPP1.

Remarks

To display the form in the application:

  1. Select Inventory --> Price Lists --> Special Prices --> Special Prices for Business Partners.
  2. Double-click a line number in the Special Prices for Business Partners table.
