Special Prices Quantity Areas Object

Description

SpecialPricesQuantityAreas is a child object of SpecialPricesDataAreas object and represents special prices that are valid only for specified quantities and above.

Source table: SPP2.

Remarks

To display the form in the application:

  1. Select Inventory --> Price Lists --> Special Prices --> Special Prices for Business Partners.
  2. Double-click a line number in the Special Prices for Business Partners table.
  3. Double-click a line number in the Special Prices for Periods table.
