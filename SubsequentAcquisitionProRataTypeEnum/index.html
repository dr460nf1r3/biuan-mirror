Subsequent Acquisition Pro Rata Type Enum Enumeration

Description

Specify one of the PR Temporis Type to determine the depreciation start date.

Members

MemberValueDescription
saprtExactlyDailyBase0Recalculates the asset depreciation from the very day when the subsequent acquisition takes place.
saprtFirstDayOfCurrentPeriod1Recalculates the asset depreciation from the first day of the posting period within which the subsequent acquisition takes place.
saprtFirstDayOfNextPeriod2Recalculates the asset depreciation from the first day of the posting period that follows the period of subsequent acquisition.
