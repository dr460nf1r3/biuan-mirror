API Tax Extension
Tax Extension Object

Description

TaxExtension is a child object of the Documents object. It stores fiscal IDs of marketing documents. Every marketing document has a set of fiscal IDs.

The TaxExtension object is applicable for cluster B (country-specific for Brazil and India).

Source tables: CPI12, CPV12, CSI12, CSV12, DLN12, DRF12, IGE12, IGN12, INV12, PCh22, PDN12, POR12, QUT12, RDN12, RDR12, RIN12, RPC12, RPD12, and WTR12.

Remarks

To display the form in the application:

    1. Select a marketing document.
    2. Select the Tax tab.
    3. Click the Fiscal IDs button.
