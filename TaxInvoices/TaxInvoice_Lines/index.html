Tax Invoice Lines Object

Description

TaxInvoice_Lines is a child object of the TaxInvoices object that represents the line entries of each tax invoice document.

Source table: TSI1 for sales invoices, TPI1 for purchase invoices, or TXD1 for journal entry.
