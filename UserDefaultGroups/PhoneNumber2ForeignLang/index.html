Phone Number 2 Foreign Lang Property

Description

Sets or returns the second default phone number in foreign language.

Field name: FrgnPhone2.

Length: 20 characters.

Property type

Read-write property

Syntax

Public Property PhoneNumber2ForeignLang() As String

Remarks
