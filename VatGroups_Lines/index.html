API Vat Groups Lines
Vat Groups Lines Object

Description

VatGroups_Lines is a child object of the VatGroups_Lines object. It enables to define the tax percentage and the effective from date for the tax group.

Source table: VTG1.

Object Model

VatGroups_Lines
UserFields
Fields
Field
ValidValuesValidValue

Remarks

Country-specific for Europe.

Mandatory property: Effectivefrom.

To display the form in the application:

  1. Select Administration --> Setup --> Financials --> Tax --> Tax Groups.
  2. Click the Tax Definition button.
